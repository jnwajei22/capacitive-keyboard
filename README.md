# Capacitive Keyboard

Custom capacitive keyboard project including hardware, firmware, and supporting tools.

## Repo Structure

- `docs/` — design notes, architecture, build logs, manufacturing notes
- `firmware/` — embedded firmware and utilities
- `hardware/` — CAD, PCB design files, prototypes, simulations
- `photos/` — build progress and testing images
- `refs/` — reference material, datasheets, external resources
- `software/` — desktop-side tools, calibration scripts, visualization
- `templates/` — reusable templates for BOMs, logs, and test plans

## Current Goals

- Define keyboard layout and sensing architecture
- Develop PCB and controller design
- Build firmware for capacitive key scanning and HID output
- Prototype enclosure and mounting system
